# IdeaverseLIFE

An [Obsidian](https://obsidian.md/) vault template structured around the **Ideaverse** philosophy, designed to serve as the knowledge backbone of a personal LIFE repository managed through GitHub.

## Background

This repository was created after learning about the concept of managing life with GitHub — specifically the TIL (Today I Learned) and LIFE repository approach. The idea is to use GitHub not just for code, but as a personal operating system: a place to capture learning, track decisions, and build a long-term record of your life and thinking.

The LIFE repository is paired with a [TIL repository](https://github.com/x5gtrn/TIL) and tracked through a GitHub Project. Rather than using a generic folder structure, this vault is built on the **Ideaverse** methodology by [Nick Milo](https://www.linkingyourthinking.com/), which prioritizes original thinking, plain-text longevity, and emergent organization over rigid hierarchies.

## Structure

This vault follows the **ACE + X** framework:

```
IdeaverseLIFE/
├── */              # Home, Home Basic, Ideaverse Map
├── +/              # Timeless ideas, concepts, and knowledge notes
├── Atlas/          # Timeless ideas, concepts, and knowledge notes
├── Calendar/       # Time-based notes (daily notes, journals, logs)
├── Efforts/        # Time-bound work (active projects, tasks)
└── x/              # Everything else (attachments, templates, samples)
    └── Ideaverse Lite 1.5/   # Bundled starter kit (removable)
```

## Key Features

### Ideaverse Lite 1.5 Included

The full [Ideaverse Lite 1.5](https://www.linkingyourthinking.com/) starter kit is bundled under `x/Ideaverse Lite 1.5`. Its sample notes and links are fully wired up — open the **Graph view** in Obsidian to see the mapping in action and get a feel for how a mature Ideaverse vault looks.

### Clean Slate on Demand

Once you're ready to use this as your own vault, simply delete `x/Ideaverse Lite 1.5`. The ACE + X scaffold remains. The **File Cleaner Redux** plugin (pre-installed) will then automatically remove any orphaned attachments and empty files left behind, leaving you with a clean starting point.

### AI-Ready with `CLAUDE.md`

A `CLAUDE.md` file is included at the root to guide [Claude Code](https://claude.ai/code) when working inside this vault. It encodes the Ideaverse principles — ACE + X categorization, no over-foldering, and a strict separation between original human thinking and AI-generated content — so that Claude can assist with vault management without disrupting your personal knowledge system.

## Getting Started

1. Clone or download this repository.
2. Open the folder as a vault in Obsidian.
3. Explore `x/Ideaverse Lite 1.5` and check the Graph view to understand the structure.
4. When ready to start fresh, delete `x/Ideaverse Lite 1.5` and let File Cleaner Redux clean up.
5. Begin adding your own notes into `Atlas`, `Calendar`, and `Efforts`.

## Philosophy

> "Bet on text. Keep it simple. Let structure emerge."

- **Plain text first** — Markdown files remain readable across decades, tools, and platforms.
- **Big buckets over rigid folders** — Start broad. Add structure only when patterns naturally appear.
- **Original thinking is sacred** — The vault exists to serve your mind, not to be organized for its own sake.
